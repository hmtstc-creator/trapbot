⚠ Grid Bot nedir?
Fiyat bir aralıkta gidip gelirken otomatik al-sat yapar. Fiyat aralık dışına çıkarsa zarar edebilir. Başlangıçta küçük bütçe (50-100 USDT) ile test et.

⚠ Futures SHORT nedir?
Fiyatın düşeceğini düşünüyorsun → SHORT açıyorsun → fiyat düşünce kâr, yükselirse zarar.
Kaldıraç riski: 3x kaldıraçta %33 aleyhine hareket = tasfiye (tüm para gider).
Başlangıçta max 3x kullan, risk %1'de tut.
🔴 TRAP FUTURES AYARLARI
Uzun zaman dilimi = daha güvenilir sinyal
⚠ MAX 5x — başlangıç için 2x veya 3x
Bakiyenin %kaçını risk alıyorsun? (1-2% önerilir)
"Analiz Et" ile Bearish Sell Trap sinyali kontrol edilir. Sinyal varsa "SHORT Aç" butonu aktif olur.

📖 BU DOSYAYI NASIL KULLANIRSIN?
ADIM 1 — Backend URL'ini Yaz
Bu HTML dosyasının en üstünde şu satırı bul:
const API = "https://RENDER_URL_BURAYA_YAZ";
Render.com'dan aldığın URL'yi tırnaklar içine yaz. Bitti.
ADIM 2 — Ana Siteye Nasıl Eklersin?
Bu dosyayı frontend/public/ klasörüne at.
Sonra tarayıcıda: siteadresi.vercel.app/besbot.html
Veya ana sitene bir link ekle: <a href="/besbot.html">BesBot</a>
ADIM 3 — Testnet Futures API Key
Futures testnet için farklı bir adres var:
testnet.binancefuture.com
Buradan key al → Render'daki environment variables'a ekle.
Spot ve Futures için aynı key çalışır (testnet.binance.vision).
⚠ GERÇEK PARAYA GEÇİŞ
Render Environment Variables'da:
BINANCE_TESTNET = false
Yap. Ve binance.com'dan gerçek API key al.
⚠ Gerçek kaldıraçlı işlem = hızlı para kaybı riski.
İlk ay sadece testnet kullan. Stratejini test et.
